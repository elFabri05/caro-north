# Images Folder

Place your images here and reference them in your CSS.

## Suggested Image Names:
- `hero-1.jpg` - First full-screen background image
- `hero-2.jpg` - Second full-screen background image

## How to Use:
After placing your images here, update the background URLs in:
`src/app/page.module.css`

Example:
```css
background: url('/images/hero-1.jpg');
```
